# Post-Launch Optimization Checklist

Time window: 30 days after launch

## Performance Tuning

- Review p95/p99 latency by endpoint
- Optimize slow DB queries and indexes
- Tune cache settings (if applicable)

## Incident Review

- Review Sev-1/Sev-2 incidents
- Identify top recurring root causes
- Implement preventive fixes

## User Feedback

- Analyze feedback submissions
- Prioritize top friction points
- Schedule follow-up experiments
