## Create SmartMeter entity

```bash
curl -i -X POST "http://localhost:9090/ngsi-ld/v1/entities" \
  -H "Content-Type: application/ld+json" \
  -H "NGSILD-Tenant: alpha" \
  --data-binary @docs/ngsi-ld/examples/entity-smartmeter.jsonld
```

## Create SmartMeterMeasurement entity

```bash
curl -i -X POST "http://localhost:9090/ngsi-ld/v1/entities" \
  -H "Content-Type: application/ld+json" \
  -H "NGSILD-Tenant: alpha" \
  --data-binary @docs/ngsi-ld/examples/entity-measurement.jsonld
```

## Query measurement

```bash
curl -s "http://localhost:9090/ngsi-ld/v1/entities?type=SmartMeterMeasurement" \
  -H "Accept: application/ld+json" \
  -H "NGSILD-Tenant: alpha"
```
