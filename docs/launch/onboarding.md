# Onboarding Guide

This guide helps new users get started with FreeFlow.

## Quick Start

1. Sign in with your organization account.
2. Land on the dashboard and review widgets.
3. Open the Alarms module and acknowledge an alert.
4. Open the Inbox and complete a task.

## Key Workflows

- **Dashboards:** personalize layout and widget visibility.
- **Alarms:** review alerts, acknowledge, and assign.
- **Inbox:** view tasks, update status, and add notes.
- **Mini Apps:** open a mini app for detailed workflows.

## Troubleshooting

- If login fails, verify account access and token expiry.
- If widgets do not load, refresh and check network status.
- If alarms are missing, confirm service connectivity.
